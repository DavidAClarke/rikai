An AI-built tool to help learn a topic based on material the user provides. Importantly, the goal is for developing a deeper _understanding_ of a topic, not just about helping with recall. For example, one can upload a scientific article and then be given a variety of questions to test your understanding of the paper.

**Important**

When visiting the URL for the first time, you will see the
API key entry screen. Each person needs their own Anthropic API key.

To get an API key:
1. Go to console.anthropic.com
2. Sign up for a free account
3. Navigate to API Keys → Create Key
4. Copy the key (it starts with sk-ant-)

API keys are free to create. Users only pay for what they use —
typically a few cents per study session.

New accounts receive $5 in free credits, which covers many sessions
before any payment is needed.
